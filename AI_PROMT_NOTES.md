# Нотатки щодо токенів AI-промпту

## Мета перевірки

Визначити орієнтовну кількість токенів, яку використовує поточний промпт фінансового аналізу для Gemini LLM.

## Інструмент і метод

- Бібліотека: `tiktoken` 0.13.0.
- Кодування: `cl100k_base`.
- Перевірявся промпт `promts/financial_analysis_gemini.md`, сформований функцією `build_financial_analysis_prompt`.
- Вхідні дані: тестовий агрегований JSON.
- Для оцінки відповіді використано типовий валідний JSON з усіма чотирма полями, які очікує API.

## Висновок

Основну частину токенів у цьому тесті становлять інструкції промпту. Кількість токенів змінюватиметься залежно від кількості категорій, довжини назв і тексту відповіді.

`tiktoken` не є офіційним токенізатором Gemini, тому наведені значення використовуються для порівняльної оцінки та контролю зростання промпту. Точне використання токенів Gemini слід отримувати з метаданих відповіді Gemini API, якщо вони доступні для обраної моделі.
